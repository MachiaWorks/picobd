# build 

## manufacturing PCB/Plates

use "pcb.zip","plate.zip",and "bottom.zip" in "/gerber".

I usually order from JLCPCB in manufacturing.

## required parts

* Raspberry Pi Pico
* Keyswitch * 38
* Keycap * 38
* M2 8mm Screw * 10
* M2 3mm Screw * 10
* M2 3mm spacer * 10
* M2 5mm spacer * 10
* gom ashi
* USB micro cable
